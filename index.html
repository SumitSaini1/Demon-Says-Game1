📖 How to Play

  • Press any key to start the game.
  • The game will flash a button (color) — remember it!
  • Click the same button to repeat the sequence.
  • Each new level adds one more color to the sequence.
  • If you make a mistake, the game will end and show your score.
  • Press any key again to restart.
